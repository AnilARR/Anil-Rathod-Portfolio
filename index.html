About

.NET Engineer

I am a Senior Software Engineer (.NET) with 5+ years of experience in designing, developing, and maintaining secure, scalable, and high-performance enterprise applications across banking, insurance, and financial domains.

I specialize in C#, .NET Core, .NET Framework, ASP.NET Web API, and Angular full-stack development, with strong expertise in building RESTful APIs, enterprise backend systems, and scalable distributed applications. I have hands-on experience delivering end-to-end solutions including API development, UI integration, system enhancements, and production support in enterprise environments.

I have strong experience in SQL Server, Oracle, PostgreSQL, and Sybase, along with SSIS and SSRS, working on ETL workflows, stored procedures, query optimization, and financial data processing systems.

Currently, I work at Intellias as a Senior .NET Engineer, supporting a UK-based banking client (BLME), contributing to mission-critical systems using .NET Core, Web API, Azure Cloud, SSIS, and SSRS. Responsibilities include backend development, ETL processing, reporting, production support, and integration with Dynamics 365 APIs. I also work with SQL Server, Sybase, GitLab, IIS, and PowerShell.

At Pinnacle Technologies, I worked on the ICICI Lombard BANCA project as a .NET Full Stack Developer — developing enterprise applications using .NET Core 8 and .NET Framework 4.8, implementing JWT authentication, AES and RSA encryption, working on Kafka-based event-driven architecture, API integrations, and migration from .NET Framework to .NET Core.

At Paramatrix Technologies, I worked on the CRISP Rating System (Client: ICRA) as a Full Stack Developer, contributing to Angular, .NET Web API, SQL Server, CI/CD using GitHub Actions, AWS deployment, OAuth 2.0 authentication, and SSO. I also worked on the Kotak Mahindra Bank project handling API development, Angular modules, SSIS ETL pipelines, SQL Server & Sybase stored procedures, BODS integration, and JWT-based security.

I have strong exposure to Azure Cloud, Docker, CI/CD pipelines (Azure DevOps, Jenkins, GitHub Actions), Kafka, and PowerShell automation. I am open to opportunities in .NET Backend Development, Full Stack .NET (Angular), Azure Cloud Engineering, and Enterprise Architecture roles.
